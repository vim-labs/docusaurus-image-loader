# docusaurus-image-loader

This plugin for Docusaurus v2 inlines small (<8KB) images and copies larger media.

## Installation:

```bash
yarn add docusaurus-image-loader # Or, npm install docusaurus-image-loader
```

## Usage:

```javascript
// docusaurus.config.js

module.exports = {
  ...
  plugins: [
    ["docusaurus-image-loader"]
  ],
  ...
}
```
